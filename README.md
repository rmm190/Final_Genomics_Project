# Final_Genomics_Project
BIOL-3615 Group Project

# What is the purpose of this project?
As part of this bioinformatics workflow, we analyzed fecal microbiome samples from Control, Mild, and Severe COVID-19 groups to better understand how gut microbial diversity and composition vary with disease severity. Our goal was to reproduce and adjust the analytical approach used by Li et al. (2021) in order to both gain familiarity with common bioinformatics tools and evaluate patterns in microbial diversity associated with COVID-19 infection. Through this workflow, we were able to explore how gut microbiome diversity and composition shift across COVID-19 severity groups.
